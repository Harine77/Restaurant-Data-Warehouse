# Restaurant Data Warehouse

Streamlit analytics dashboard for the restaurant data warehouse project.

## Run locally

```bash
streamlit run 10_dashboard.py
```

## Deploy on Streamlit Community Cloud

1. Push this repository to GitHub.
2. Open Streamlit Community Cloud.
3. Create a new app from the GitHub repository.
4. Set the main file path to `10_dashboard.py`.
5. Store production database credentials outside the repo, ideally in Streamlit secrets.
